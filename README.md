#  Dynamic Portfolio Web Application

This is a full-stack personal portfolio website for **Priyanshi Lohani**, built with a **Next.js frontend** and a **Flask backend**. The site dynamically showcases skills, projects, and research papers â€” perfect for presenting your professional profile online.

---

##  Features

-  Dynamic **Projects** and **Research Papers** sections (served via API)
-  Responsive layout with clean UI
-  Downloadable resume
-  Contact links to GitHub, LinkedIn, and Email

---

##  Tech Stack

### Frontend
- **Next.js** 
- CSS modules
- Hosted on **Vercel**

### Backend
- **Flask** (Python)
- RESTful APIs for serving projects, programs, research paper data and Resume
- Hosted on **Render**
